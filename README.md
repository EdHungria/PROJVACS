# PROJVACS - Sistema de Gerenciamento de Vacinas para Animais

Projeto desenvolvido na disciplina de Programação Orientada a Objetos e Desenvolvimento em Java, ministrada pelo **Prof. Fábio Bezerra** na Universidade Federal Rural da Amazônia (UFRA).

## Descrição

Aplicação web para gerenciamento de animais de estimação e suas vacinas, com controle de usuários (Administrador, Veterinário e Usuário comum).

O sistema permite:
- Cadastro, listagem, edição e exclusão de animais
- Associação de raças
- Registro de vacinas aplicadas
- Controle de acesso por perfil de usuário

## Tecnologias Utilizadas

- **Java 17**
- **Jakarta EE / Java EE**
- **JSF 2.2** (Mojarra)
- **PrimeFaces 7.0** (interface rica)
- **JPA** com **EclipseLink 2.7**
- **MySQL** como banco de dados
- **Maven** para gerenciamento de dependências
- **Apache Tomcat 9** como servidor de aplicação
- **IntelliJ IDEA Ultimate** como IDE
